Getting started

Up and running in 15 minutes

Everything you need to go from signup to a live stream with professional overlays.

Step 1

Check your email

After payment you receive three links. Save them somewhere easy to find — you will use them before every game.

  • W
    Game Setup Wizard

    Run this before every game to enter team names, lineups, and kick-off time. The wizard sends everything to your overlays automatically.

  • C
    Controller link

    Open this on your phone or tablet during the game. Controls the scoreboard, timer, goals, fouls, and overlays in real time. Works on any device, any network.

    Bookmark it on your phone before match day.

  • O
    Overlay URL

    Paste this into OBS or Streamlabs as a Browser Source. You add it once and never need to change it.

Keep these links safe. Anyone with your controller link can manage your live scoreboard. Lost them? Write to info@futsalplay.live and we will resend them immediately.
Step 2

Add overlays in OBS or Streamlabs

Your overlay URL covers all overlay types. Add it as a Browser Source once per overlay, changing the last part of the URL for each one.

Source name URL ending Size Suggested scene
Scoreboard change /overlay to /scoreboard 1920×1080 Main / Live
Lower Third /lowerthird 1920×1080 Main / Live
Lineup /lineup 1920×1080 Main / Live
Starting Soon /startingsoon 1920×1080 Pre-match
BRB /brb 1920×1080 Break
  • 1
    Add a Browser Source

    In OBS click + under Sources and choose Browser Source. In Streamlabs it is the same under Add Source.

  • 2
    Paste your overlay URL

    Take the overlay URL from your email and change the last part to match the overlay type (see table above). Set width to 1920 and height to 1080.

  • 3
    Make the background transparent

    Paste this into the Custom CSS field in the Browser Source settings:

    body { background-color: rgba(0, 0, 0, 0) !important; }

  • 4
    Repeat for each overlay type

    You need one Browser Source per overlay. All five can live in a single OBS scene — visibility is controlled from your phone via the controller.

Step 3 — optional

Install the FOGIS bookmarklet

For clubs registered on minfotboll.se. Loads team rosters, player names, and match info into your overlays automatically with one click. Skip this if your club does not use FOGIS.

  • 1
    Open the Game Setup Wizard in Chrome

    Use the wizard link from your email. Open it on the stream PC in Google Chrome.

  • 2
    Choose FOGIS in Step 1

    Select Yes, on FOGIS. The bookmarklet installation instructions appear on the next screen. Follow them to save the bookmarklet to your Chrome bookmarks bar. Do this once.

  • 3
    Use it on match day

    Log in to minfotboll.se, open the game page, and click the bookmarklet. Team names, logos, lineups, and venue info all load into your overlays instantly.

Step 4

Match day checklist

Five things to do before kick-off, in order.

  • 1
    Run the Game Setup Wizard

    Enter team names, lineups, and kick-off time. Takes 3 to 4 minutes. If you use FOGIS, click the bookmarklet on the game page — the wizard fills in everything automatically.

  • 2
    Open the controller on your phone

    Use the controller link from the email. Bookmark it so it is one tap away on match days. No app or login required.

  • 3
    Test your upload speed

    Run a speed test at speedtest.net on the hall's Wi-Fi. Set your OBS bitrate to 70% of the result. 4 Mbps upload gives a reliable 720p30 stream at 2500 kbps.

  • 4
    Show Starting Soon while waiting for kick-off

    Tap Starting Soon in the Overlays card on the controller. It displays a live countdown, team logos, league name, and venue.

  • 5
    At kick-off

    Hide Starting Soon, confirm the period (1st Half), start the timer. From this point manage everything from the controller.

Reference

Using the controller

Everything you can do during the game from your phone or tablet.

Overlays

Tap any row to show or hide that overlay instantly. No need to switch scenes in OBS.

Timer

Start, stop, reset, and adjust. Use Adjust to correct drift against the hall's official clock mid-period.

Goals and fouls

Tap + Goal or + Foul per team. Undo is one tap away. Fouls reset automatically when you change period.

Lower Third

Set the announcement text and duration, then tap Show. Tap player names in the quick-pick list for instant goal announcements.

Red cards (2-minute)

Enter the player name, tap Home or Away. The countdown appears on the scoreboard and clears automatically on the next opposing goal.

Half time

Tap Half Time to stop the timer and display HALF TIME on screen. Tap 2nd Half when play resumes — this resets fouls and clears the label.

Keyboard shortcuts when the controller is open on a laptop: Space starts and stops the timer, 1 and 2 add goals, Q and W undo goals, F and G add fouls.
Help

Troubleshooting

Quick answers to the most common issues.

Overlays not updating in OBS
Right-click the Browser Source and choose Refresh. Make sure the URL includes your token (the long string after ?token=).
Controller says "reconnecting"
Reload the controller page. Make sure you opened the full URL from the email including the token. The controller requires an active internet connection.
Overlay background is black, not transparent
In the Browser Source properties, paste body { background-color: rgba(0, 0, 0, 0) !important; } into the Custom CSS field and click OK.
I lost my links
Write to info@futsalplay.live from the email address you signed up with and we will resend them right away.
FOGIS bookmarklet gives an error
Make sure you are logged in to minfotboll.se and on the actual game page. The URL must contain a 6-digit game ID. The bookmarklet must be installed from your wizard link, not from a saved HTML file.
Timer is running but the scoreboard shows the old time
Refresh the Browser Source in OBS. The overlay reconnects automatically after a refresh.
Something else is not working
Write to info@futsalplay.live with a short description of what is happening and we will sort it out.
