Getting started

Up and running in 15 minutes

Everything you need to go from signup to a live stream with professional overlays.

1

Check your email

After payment you receive three links
  • W
    Game Setup Wizard — start here before your first game. Enter your team names, lineups, and match info. You can run this again before every game.
    Save this link somewhere easy to find.
  • C
    Controller link — open this on your phone or tablet on match day. This is what you use to control the scoreboard, timer, goals, and overlays during the game.
    Works on any device, any network. No app to install.
  • O
    Overlay URL — paste this into OBS or Streamlabs as a Browser Source. It contains your scoreboard, lineup, lower third, and all other overlays in one URL.
    You will add this once in OBS and never need to change it.
Keep these links safe. Anyone with your controller link can manage your live scoreboard. If you ever lose them, write to info@futsalplay.live and we will resend them.
2

Add overlays in OBS or Streamlabs

Add one Browser Source per overlay

Your overlay URL covers all overlay types. You add it multiple times with different names — each instance knows which overlay to show based on how you named it in OBS.

Name in OBS URL to paste Size Scene
Scoreboard Your overlay URL (from email) — change /overlay to /scoreboard 1920×1080 Main / Live
Lower Third Same URL, change ending to /lowerthird 1920×1080 Main / Live
Lineup Same URL, change ending to /lineup 1920×1080 Main / Live
Starting Soon Same URL, change ending to /startingsoon 1920×1080 Pre-match
BRB Same URL, change ending to /brb 1920×1080 Break
Browser Source settings for each source
  • 1
    In OBS, click + under Sources and choose Browser Source.
  • 2
    Paste your overlay URL into the URL field. Change the last part to match the overlay type (scoreboard, lowerthird, etc.).
  • 3
    Set Width to 1920 and Height to 1080.
  • 4
    Paste this into the Custom CSS field to make the background transparent:
    body { background-color: rgba(0, 0, 0, 0) !important; }
  • 5
    Click OK. Repeat for each overlay type.
You do not need to switch OBS scenes to control overlay visibility. All overlays are toggled directly from the controller on your phone — just keep all browser sources active in your main scene.
3

Install the FOGIS bookmarklet (optional)

For clubs registered on minfotboll.se

If your games are listed on minfotboll.se (the Swedish Football Federation system), the bookmarklet lets you load team rosters, player names, and match info into your overlays automatically in one click. Skip this section if your club does not use FOGIS.

  • 1
    Open your Game Setup Wizard link (from the email) in Google Chrome on the stream PC.
  • 2
    In Step 1 of the wizard, choose Yes, on FOGIS. The bookmarklet installation instructions appear on the next screen.
  • 3
    Press Ctrl+Shift+B to show the bookmarks bar, then follow the on-screen steps to save the bookmarklet. Do this once.
  • 4
    On match day: log in to minfotboll.se, go to the game page, and click the bookmarklet. The wizard (or controller) updates instantly with team names, logos, lineups, and venue info.
4

Match day checklist

  • 1
    Open the Game Setup Wizard on the stream PC. Enter team names, lineups, and kick-off time. The wizard takes 3-4 minutes and sends everything to the overlays automatically.
  • 2
    Open the Controller on your phone. Use the controller link from the email. Bookmark it so it is one tap away on match day. No app or login required.
  • 3
    Test your upload speed at speedtest.net on the hall's Wi-Fi. Use 70% of the result as your OBS bitrate. 4 Mbps upload gives a solid 720p30 stream at 2500 kbps.
  • 4
    Show the Starting Soon overlay from the controller while waiting for kick-off. It displays a live countdown, team logos, league, and venue.
  • 5
    At kick-off: hide Starting Soon, confirm the period (1st Half), start the timer. From now on manage everything from the controller.
5

Using the controller during the game

Overlays
Tap any row to show or hide that overlay instantly. No need to touch OBS.
Timer
Start, stop, reset. Use Adjust to correct drift against the hall's official clock.
Goals and fouls
Tap + Goal or + Foul per team. Undo is always one tap away. Fouls reset automatically at half time.
Lower Third
Set the announcement text, choose a duration, tap Show. Tap player names in the quick-pick list for goal announcements.
Red cards (2-min)
Enter the player name, tap Home or Away. The countdown appears on the scoreboard and clears on the next opposing goal.
Half time
Tap Half Time to stop the timer and show HALF TIME on screen. Tap 2nd Half when play resumes.
Keyboard shortcuts (when the controller page is focused on a laptop): Space starts/stops the timer, 1 and 2 add goals, Q and W undo goals, F and G add fouls.
6

Troubleshooting

Overlays not updating in OBS
Right-click the Browser Source in OBS and choose Refresh. Make sure the URL is correct and includes your token (the long string after ?token=).
Controller says "reconnecting"
Reload the controller page. Make sure you opened the full URL from the email including the token. The controller requires an active internet connection.
Overlay background is black, not transparent
In the Browser Source properties, paste body { background-color: rgba(0, 0, 0, 0) !important; } into the Custom CSS field and click OK.
I lost my links
Write to info@futsalplay.live from the email you signed up with and we will resend them immediately.
Timer is running but the scoreboard still shows the old time
Refresh the Browser Source in OBS. The overlay reconnects to the server automatically after a refresh.
FOGIS bookmarklet gives an error
Make sure you are logged in to minfotboll.se and on the actual game page (the URL contains a 6-digit game ID). The bookmarklet must be installed from your wizard link, not from a saved HTML file.
Something else is not working
Write to info@futsalplay.live with a short description of what is happening and we will sort it out.
