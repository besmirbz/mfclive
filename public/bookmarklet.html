⚽ MFCLIVE — Bookmarklet Setup

One-click roster import from minfotboll.se  ·  set up once, use forever

Step 1 — Set your server URL

Use http://localhost:3000 if you'll always run the bookmarklet from the streaming PC.
Use your Cloudflare tunnel URL (e.g. https://controller.futsalplay.live) to run it from any device.

Step 2 — Copy the bookmarklet
Step 3 — Add to bookmarks bar
a
Show bookmarks bar: Ctrl+Shift+B in Chrome
b
Right-click the bookmarks bar → Add page
c
Set Name to ⚽ Load Roster and paste the copied code into the URL field → Save
🟢 Every game day — 3 steps
1
Start the server: node server.js on the stream PC
2
Log in to minfotboll.se and open the game page
3
Click ⚽ Load Roster in your bookmarks bar — a green banner confirms the load. Done.
