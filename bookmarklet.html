Create a new bookmark manually

Right-click anywhere on the bookmarks bar → Add page (or "Add bookmark")
Set the Name to ⚽ MFC Load Roster
Set the URL field to the copied code → Save

Every game day

  1. Start the server: node server.js on the stream PC
  2. Log in to minfotboll.se and open the game page
  3. Click ⚽ MFC Load Roster in your bookmarks bar
  4. A blue banner appears briefly, then a confirmation tab opens and closes
  5. Overlays and controller update automatically ✓

The confirmation tab may be blocked by Chrome's popup blocker the first time — if so, click "Allow" in the address bar and retry.
